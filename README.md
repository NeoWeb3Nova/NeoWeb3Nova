大家好，我是 **Neo Web3（Web3 的尼奥）**，一名专注链上 AI 与去中心化金融底层逻辑的开发者。

我信奉用**第一性原理**拆解复杂系统，用**费曼学习法**把晦涩的概念变成可执行代码。日常在 EVM 兼容链（Ethereum、Polygon、Monad 等）上写 Solidity，在 React / TypeScript 前端里接 Web3，也在 Rust 和 AI Agent 的交叉地带寻找新范式。

> *"Mastering Web3 with the Feynman method, building the future one block at a time."*

- 参加过 **Chainlink Hackathon**，探索去中心化预言机与智能合约的深度集成

- 相信**反脆弱**与**长期主义**：熊市筑基，牛市交付
